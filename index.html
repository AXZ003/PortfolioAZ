    WORK

    Style My Travel
    Style My Travel is a full-stack React app where jetsetters can get some seriously fab inspiration for their travels. Store your trips in a calendar, create itineraries that are saved in Mongo, & receive fab fashion advice through the Tumblr + Open Weather Map API.
    Finanza
    Finanza offers users the ability to create a personalized financial portfolio. It allows users to create an account where they can keep track of all their stocks and evaluate their current state in the market as well as to research specific stocks with up to date data.
    Countries Collector
    This app allows the user to click an image that generates a random number. The random number is updated to "Your Total Score", the user has to match their total score to the "Number to Guess". Every time the page is refreshed the "Number to Guess" will be reset and the images will reset their numbers as well.
    LitFlix
    This application allows users to search for current and past movies using the OMDB, YouTube and GraceNote API. After the user inputs a movie in the serach box the app will render the summary, rating, movie poster, trailers and showtimes for that specific movie.
    Harry Potter Memory Game
    This Harry Potter Memory game shows 12 different magical objects. Each time the user clicks an image, the magical objects shuffle. The end goal of the game is to click a new magical object each time but without clicking the same one twice! In this react-app, the goal was to break the UI into several components, manage component state, and respond to user events using React.
    Favorite Celebs
    This site calls the GIPHY API and populates the page with your favorite celebrity gifs. The user can also use this app to search for their own celebrity gifs.
